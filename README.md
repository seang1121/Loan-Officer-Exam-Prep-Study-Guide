## 🔑 Key Numbers to Know

| Item | Value |
|---|---|
| Passing score | 75% (86/115 scored questions) |
| Test time | 190 minutes |
| Questions | 125 total (115 scored, 10 unscored) |
| RESPA Good Faith Estimate → replaced by | Loan Estimate (LE) |
| HUD-1 → replaced by | Closing Disclosure (CD) |
| TILA APR tolerance (regular) | 0.125% |
| TILA APR tolerance (irregular) | 0.25% |
| HOEPA high-cost threshold (rate) | APR > 6.5% above APOR (1st lien) |
| HOEPA high-cost threshold (fees) | > 5% of loan amount |
| ATR/QM DTI ratio | ≤ 43% (general) |
| MLO compensation max (LO Comp Rule) | Cannot be based on loan terms |
| 3-day TRID waiting period | After LE delivery before consummation |
| 3-day rescission right | After consummation (refinances/2nd mortgages) |
| HMDA LAR filing deadline | March 1 of following year |
| SAR filing deadline (BSA) | 30 days after suspicious activity detected |
| USA PATRIOT Act — CIP requirement | Verify identity before opening account |
